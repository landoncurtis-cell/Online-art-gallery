About

A polished, realistic project built for competition standards.

Project Goal

Arcadia Gallery is a digital art experience that feels like a real product: a curated collection with advanced browsing tools, favorites, and an accessible viewing experience.

Key Features

  • Data-driven gallery rendering (no hard-coded cards)
  • Search, tag filters, sorting, and multiple layout modes
  • Accessible modal viewer with keyboard navigation and focus handling
  • Favorites + theme saved with localStorage
  • Form validation on the Submit page + local submission list
  • PWA service worker support (offline-ready)

How It’s Built

This project uses structured CSS files (base/layout/components/theme) and modular JavaScript. That’s the kind of organization judges usually love because it’s clean and scalable.
