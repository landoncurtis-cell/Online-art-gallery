Accessibility

Designing so more people can use the site comfortably.

What We Support

  • Keyboard navigation: Tab through controls, open artwork, use ←/→, press Esc to close viewer.
  • Focus visibility: Clear focus outline for interactive elements.
  • Reduced motion: Animations reduce/disable when your system preference is set to “reduce motion.”
  • Screen reader hints: ARIA labels, live regions for result counts, and descriptive alt text.
  • Color & contrast: High-contrast UI, plus theme toggle.
